# \## 🚀 OVERVIEW

# This repository features a production-ready \*\*Retrieval-Augmented Generation (RAG)\*\* system specifically designed for financial document analysis. By leveraging the power of \*\*Gemini 2.5 Flash\*\* and \*\*FAISS vector storage\*\*, it allows users to perform complex queries against financial reports with high precision and low latency.

# 

# \## 💡 Key Technical Features

# \* \*\*Next-Gen LLM:\*\* Integrated with the latest \*\*Google Gemini 2.5 SDK (2026)\*\*.

# \* \*\*Privacy-Centric RAG:\*\* Document embeddings are processed locally using `sentence-transformers`, ensuring that only relevant context snippets are sent to the cloud.

# \* \*\*Efficient Retrieval:\*\* Uses \*\*FAISS (Facebook AI Similarity Search)\*\* for high-performance semantic indexing.

# \* \*\*Clean Architecture:\*\* Built with a modular approach, making it easy to scale for multi-document analysis or different LLM providers.
